Git

  • git clone — скачивает удалённый репозиторий и создаёт его локальную копию
  • git status — отображает актуальное состояние файлов в рабочем каталоге и индексе
  • git log — отображает список коммитов в порядке от самых новых к более старым, позволяя проследить историю изменений проекта
  • git add — добавляет файлы в индекс
  • git commit — добавляет файлы в репозиторий
  • git push — передаёт сделанные локально изменения в связанный удалённый репозиторий для их сохранения и совместной работы
  • git pull — получает изменения из удалённого репозитория и автоматически сливает с локальной веткой
